# movies-explorer Backend для дипломной работы 
Репозиторий бекенда для сайта movies-kirill.nomoreparties.co

## В проекте реализовано 
* Регистрация и авторизация пользователей 
* Изменения информации о пользователе
* Добавление фильма
* Удаление фильма

## Технологии 
* Node.js
* Express
* MongoDb
* Winston
* Cookie
* Express-rate-limit
* Helmet
